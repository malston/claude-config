# CLAUDE.local.md

This file provides additional guidance to Claude Code when working with code in this repository.

## Documentation Guidelines

### Company Names in Examples

- **NEVER use real company names** in documentation, code comments, or examples
- **ALWAYS use generic placeholder names** such as:
  - ACME Corporation / ACME Corp / acme.com
  - Example Corp / example.com
  - Your Company / your-company.com
  - Generic Corp / generic.com
  - Sample Industries / sample.com

### Real Company References to Avoid

- Wells Fargo, Microsoft, Google, Amazon, IBM, etc.
- Any Fortune 500 companies
- Any identifiable real-world organizations

### Preferred Generic Examples

- Domain names: `acme.com`, `example.com`, `your-company.com`
- Repository names: `your-org/your-repo`, `acme-corp/sample-app`
- Email addresses: `user@example.com`, `admin@acme.com`
- API URLs: `https://api.acme.com`, `https://github.acme.com/api/v3`

This ensures documentation remains professional and avoids any potential trademark or branding issues.
